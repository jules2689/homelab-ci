- `##` / `###` — headings
- ` ``` ` … ` ``` ` — fenced code block (optional language)
- `**bold**` / `*italic*` — bold, italic
- `- item` / `1. item` — lists
- `[text](url)` — links
- `::: Title` … `:::` — collapsible section (rendered as `<details>` / `<summary>`); body until closing `:::`
